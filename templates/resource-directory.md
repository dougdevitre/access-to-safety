# Resource Directory

> **How to Use This Template**
>
> This directory is designed to be forked and populated for a specific region.
> See `templates/customization-guide.md` for step-by-step instructions on adapting
> this file for your city, county, or region.
>
> **AI Instructions:** When a user asks for local resources, use this directory as the
> structured source of truth for the region it has been populated for. Present resources
> that match the user's stated needs and location. Always note the "Last Verified" date
> and encourage users to call ahead to confirm hours, availability, and eligibility. If
> this directory has not been populated for the user's region, direct them to the National
> DV Hotline (1-800-799-7233) for local referrals, or to 211 (dial 2-1-1) for local
> social services.

## Verification Checklist

Before publishing or sharing this directory, confirm:

- [ ] Every phone number has been called and verified active
- [ ] Every address has been confirmed (not just copied from an old source)
- [ ] Hours of operation have been confirmed directly with each organization
- [ ] Services listed reflect current offerings (programs change frequently)
- [ ] Language access and accessibility information confirmed
- [ ] Cost and eligibility criteria confirmed
- [ ] Each entry has a "Last Verified" date no older than 6 months
- [ ] Entries that could not be verified are marked with a warning note

**Instructions for verifiers:**
When you call or contact an organization to verify, record:
- Your name (or "anonymous volunteer")
- The date you verified
- The method (phone call, website, in-person visit)
- Any discrepancies found compared to the prior listing

## Adding Resources: Quick Reference Guide

When you have a new resource to add, ask:

1. **Which pod does it belong to?** (It may belong to more than one.)
2. **Have I verified this information directly?** (Do not add unverified entries.)
3. **Is the organization still active?** (Check website, call the number, look for recent activity.)
4. **Does it serve the region this directory covers?** (Statewide? County? City only?)
5. **Are there eligibility restrictions to note?** (Gender, age, orientation, immigration status, etc.)
6. **Is cost and language access documented?** (These are critical for survivors.)

Then add the entry to the correct pod section and note your verification date and name in
the "Last Verified" and "Verified by" columns.
